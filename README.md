# 🚀 Portfólio Pessoal - João Ricardo

Este repositório contém o código-fonte do meu portfólio online. O projeto foi desenvolvido para centralizar meus projetos, habilidades técnicas e experiências profissionais, como minha trajetória na **Visteon**.

---

## 🛠️ Tecnologias e Ferramentas

Para construir esta interface moderna e responsiva, utilizei:

* **Linguagens:** HTML5, CSS3 e JavaScript.
* **Design:** Glassmorphism, Gradientes suaves e Design Responsivo (Mobile-First).
* **Animações:** CSS Keyframes e transições dinâmicas para a seção de Skills.
* **Versionamento:** Git & GitHub.

---

## ✨ Funcionalidades Principais

* **📱 Responsividade Total:** Interface adaptada para smartphones, tablets e desktops.
* **✨ Animações de Skills:** Transições suaves que destacam as competências técnicas.
* **📂 Galeria de Projetos:** * *Criador de Cartões:* Geração de crachás com QR Code.
    * *Pokedex Consulta:* Consumo de API externa para exibição de dados.
* **📄 Currículo Integrado:** Botão direto para visualização/download do currículo.
* **✉️ Seção de Contato:** Links rápidos para redes profissionais.
